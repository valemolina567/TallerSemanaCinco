# Rocket Launch Simulation

## Descripción

Rocket Launch Simulation es un proyecto desarrollado en Unity en 3D como parte de un taller enfocado en la implementación de conceptos básicos de física y el uso de Visual Scripting.

El proyecto consiste en la simulación del lanzamiento de un cohete construido con objetos básicos (cubos), junto con una plataforma de lanzamiento y una estructura de soporte. El objetivo principal fue aplicar conceptos como gravedad, colisiones y lógica de eventos sin necesidad de programar en C#.

El sistema incluye un botón llamado **Launch**, que al ser presionado activa una secuencia de acciones: primero se retraen los brazos mecánicos que sostienen el cohete mediante una animación, y posteriormente se aplica una fuerza al cohete para simular su despegue vertical.

---

## Objetivo

Aprender a implementar conceptos básicos de física en Unity, como gravedad y colisiones, además de explorar el uso de Visual Scripting para controlar la lógica del juego mediante nodos visuales.

---

## Características principales

* Escena 3D desarrollada en Unity
* Cohete y estructura creados con cubos
* Uso de prefabs para organización de objetos
* Implementación de Rigidbody en el cohete
* Sistema de colisiones entre objetos
* Animación de la estructura de soporte
* Botón interactivo de lanzamiento
* Lógica desarrollada con Visual Scripting
* Simulación de lanzamiento vertical

* Grafo de Visual Scripting de la estructura

---

## Funcionamiento

1. El cohete permanece sobre la plataforma sostenido por la estructura.
2. El usuario presiona el botón **Launch**.
3. Los brazos mecánicos de la estructura se retraen mediante una animación.
4. Se aplica una fuerza al cohete utilizando Rigidbody.
5. El cohete despega verticalmente simulando un lanzamiento real.
